# 📱 Piscine Pedago Mobile - Flutter & Dart

¡Bienvenidos al repositorio principal de la **Piscine Pedago Mobile**! Este espacio documenta un viaje intensivo de aprendizaje, desde los cimientos de la programación móvil hasta la creación de aplicaciones robustas, conectadas y profesionales utilizando el ecosistema de **Flutter** y **Dart**.

Este proyecto no es solo una colección de ejercicios, sino una progresión lógica diseñada para dominar el desarrollo cross-platform moderno. A lo largo de estas semanas, hemos transformado ideas abstractas en interfaces tangibles, datos crudos en información útil y usuarios anónimos en perfiles autenticados.

## 🗺️ El Mapa de la Piscine

Nuestra aventura se dividió en seis grandes módulos, cada uno enfocado en una competencia clave del desarrollo móvil actual:

### 🔹 [Módulo 00: Cimientos y Reactividad](./mobileModule00/)
El punto de partida. Aquí aprendimos a pensar en "árboles de widgets". Construimos desde un simple "Hola Mundo" hasta una **calculadora funcional** capaz de gestionar su propio estado y resolver operaciones matemáticas complejas.
*   **Key skills:** State Management (`setState`), Layouts (Column, Row, Center), Lógica de negocio en Dart.

### 🔹 [Módulo 01: Estructura y Navegación](./mobileModule01/)
Aprendimos a organizar el espacio. Creamos la base de nuestra **Weather App**, implementando navegación por pestañas, campos de búsqueda y una arquitectura visual profesional que se adapta a las necesidades del usuario.
*   **Key skills:** `BottomNavigationBar`, `TextField`, `AppBar` personalizada, navegación multi-pantalla.

### 🔹 [Módulo 02: El Mundo Real (APIs y GPS)](./mobileModule02/)
Damos el salto a la conectividad. Nuestra aplicación de clima dejó de ser una maqueta para consumir datos reales de la **API de Open-Meteo** y utilizar el **GPS del dispositivo** para geolocalizar al usuario.
*   **Key skills:** HTTP Requests, JSON Parsing, Geolocalización, Programación Asíncrona (`async/await`), Arquitectura por capas.

### 🔹 [Módulo 03: Diseño Avanzado y Datos](./mobileModule03/)
Transformamos datos en conocimiento. Implementamos **gráficos dinámicos** para visualizar tendencias meteorológicas y adaptamos la interfaz para que sea fluida en cualquier orientación (Portrait/Landscape).
*   **Key skills:** `fl_chart`, `TabController`, Diseño responsivo, Geocodificación inversa.

### 🔹 [Módulo 04: Ecosistema Firebase y Autenticación](./mobileModule04/diary_app/)
Entramos en el terreno de las aplicaciones con backend. Desarrollamos un **Diario Personal** con autenticación de Google y persistencia de datos en la nube.
*   **Key skills:** `Firebase Auth`, `Firestore`, `Firebase Storage`, Google Sign-In, gestión de imágenes.

### 🔹 [Módulo 05: Experiencia de Usuario Pro](./mobileModule05/advanced_diary_app/)
El broche de oro. Refinamos nuestro diario añadiendo una vista de **calendario interactivo** y optimizando la gestión de grandes volúmenes de datos y estados complejos.
*   **Key skills:** `table_calendar`, gestión avanzada de bases de datos NoSQL, pulido final de UX/UI.
